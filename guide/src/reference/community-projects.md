# Community Projects

This page showcases libraries, tools and platforms built for or with Wasm Bindgen.

## Wasm Bindgen Utility Libraries & Tools

- [console_error_panic_hook](https://github.com/rustwasm/console_error_panic_hook) - Better panic messages in the browser console
- [gloo](https://github.com/rustwasm/gloo) - Modular toolkit providing ergonomic wrappers around browser APIs
- [serde-wasm-bindgen](https://crates.io/crates/serde-wasm-bindgen) - Native integration between serde and wasm-bindgen
- [wasm-logger](https://crates.io/crates/wasm-logger) - Logger implementation for wasm-bindgen compatible with the log crate

## Platforms using Wasm Bindgen

- [Bevy](https://bevyengine.org/) - Data-driven game engine with wasm-bindgen support for browser games
- [Cloudflare Workers Rust](https://github.com/cloudflare/workers-rs) - Rust SDK for Cloudflare Workers built on wasm-bindgen
- [Dioxus](https://dioxuslabs.com/) - Fullstack framework for web, desktop, and mobile apps with a single codebase
- [Leptos](https://github.com/leptos-rs/leptos) - Full-stack framework with fine-grained reactivity and server-side rendering
- [Yew](https://yew.rs/) - React-like framework for building multi-threaded frontend web apps with WebAssembly
