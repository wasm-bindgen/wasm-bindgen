# `js-sys`

The [`js-sys` crate][js-sys] provides bindings to all the global APIs
guaranteed to exist in every JavaScript environment by the ECMAScript standard.

For documentation on js-sys types and their generic type parameters, see the
[js-sys](../../reference/types/js-sys.md) reference.

For API documentation, see [docs.rs][docsrs].

[js-sys]: https://crates.io/crates/js-sys
[docsrs]: https://docs.rs/js-sys
